AndroidTranslator is a simple application designed to facility the translation
 of Android Resource files to other languages.

The application was created as Visual Studio 2010 project wrote in C# for 
.Net 4.0.  To run the application you will need to have the .Net 4.0 Framework 
installed.


At the current time it supports translating the resource strings using the following APIs:
1. Google Translate
2. Microsoft Translate
